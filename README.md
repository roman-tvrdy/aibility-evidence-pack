# Roman Tvrdý × Aibility — evidence pack

Ahoj, tohle není klasické portfolio. Je to malý evidence pack k odpovědi pro Aibility: co jsem za poslední dva měsíce dělal s AI, co z toho funguje, co drhlo a jak nad tím přemýšlím.

## Co je tady

- `aibility-formular-copy-paste.md` — texty připravené přímo do tří polí formuláře.
- `aibility-roman-tvrdy-podklad.md` — delší pracovní podklad: projekty, reflexe, AI pohled a dataset z ladění odpovědi.
- `raw-conversation-dataset.md` — samostatný hrubý dataset konverzace při ladění odpovědi a evidence packu.
- `serverhotel-landing-pyramida.html` — funkční HTML prototyp ServerHotel diagnostiky a konfigurátoru.
- `serverhotel-prototyp.md` — bezpečné executive summary toho, co je kolem ServerHotelu připravené. Interní MD podklady neposílám veřejně.
- `pactman-cowork-brief-2026-05-16.md` — aktuální Cowork brief k PACTmanu jako systému druhého mozku pro práci člověk + AI.
- `pact-os-manual.html` — starší HTML ukázka vývoje PACTmana, aby bylo vidět, odkud se systém posouval.
- `code-squad-delivery-model-anonymized.md` — anonymizovaný delivery model pro Code Squad: metodika, gate logika, review a sign-off bez jmen, firem a repo odkazů.
- `fail-digitalni-jeskyne-14-dni.html` — ukázka výstupu pro FAIL Digitální jeskyni.
- `kill-bill-lessons-learned-prezentace.html` — Kill Bill / KB2 lessons learned prezentace.

## Co tu záměrně není

Kill Bill workflow sem nedávám jako veřejný odkaz. Funguje, ale obsahuje fakturační údaje a klientské podklady. To není věc, kterou bych měl posílat veřejně v náborovém formuláři.

ServerHotel interní MD podklady sem nedávám. Obsahují pracovní detaily, které není potřeba veřejně ukazovat. Místo toho je v repozitáři funkční HTML prototyp a bezpečné executive summary bez citlivých detailů, rozpočtů kampaní a interních rozhodovacích poznámek.

Code Squad veřejně posílám pouze v anonymizované podobě. Jména, firmy, klientský kontext a konkrétní repo odkaz jsou odstraněné.

## Rychlý orientační souhrn

Za poslední dva měsíce jsem s AI postavil nebo rozpracoval:

- funkční fakturační MVP Kill Bill,
- ServerHotel diagnostiku IT infrastruktury včetně veřejně ukázatelného HTML prototypu,
- deep research a GEO/SEO analýzu SafeDX.eu,
- PR a management podklady pro SafeDX,
- anonymizovaný Code Squad delivery model pro řízenou AI-assisted dodávku,
- PACTman systém pro kontinuální práci člověk + AI.

Nejde mi o to ukázat dokonalý profil. Jde mi o to ukázat reálný pracovní režim: rychle stavět, narazit, opravit směr, poučit se a pokračovat.
